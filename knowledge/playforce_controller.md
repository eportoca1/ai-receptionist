# EW Support Card — Play Force Wireless Controller (Electro Play Force)

## Product keywords (listen for these)
Play Force, Electro Play Force, wireless controller, game controller, "Electro Play Force" (Bluetooth name)

## Quick triage questions (ask 1 at a time)
1) What device are you connecting to? (iPhone/iPad/Mac, Android, Windows PC, Nintendo Switch)
2) Is the controller currently ON or OFF?
3) What LED color is flashing (if they see it)?
4) What exactly is happening? (won’t pair, connects then drops, buttons wrong, not charging, etc.)

## Power + pairing basics
- Controller must be OFF before pairing
- Pairing: hold the correct button combo for ~2 seconds, then connect in Bluetooth
- If pairing fails: power OFF and try again

### Power controls
- Power off: hold HOME for ~5 seconds
- Pairing combos require holding for ~2 seconds

## Pairing modes (device → buttons → LED color → Bluetooth name)
### Apple (iPhone / iPad / Mac)
- Mode: Apple Controller Mode
- Buttons: hold B + HOME for ~2 seconds
- LED: flashing Blue → solid Blue when connected
- Bluetooth name: "Electro Play Force"

Steps:
1) Controller OFF
2) Hold B + HOME ~2 seconds (LED flashes Blue)
3) On iPhone/iPad/Mac: Settings → Bluetooth → select “Electro Play Force”
4) Launch a controller-supported game

### Android (Recommended: Native controller mode)
- Mode: Android Controller Mode
- Buttons: hold X + HOME ~2 seconds
- LED: flashing Light Blue
- Bluetooth name: “Electro Play Force”

Steps:
1) Controller OFF
2) Hold X + HOME ~2 seconds (LED flashes Light Blue)
3) Android Bluetooth settings → select “Electro Play Force”
Tip: Some games require controller support enabled in the game settings.

### Android (Button mapping mode — only if game has no controller support)
- Mode: Key Mapping Mode
- Buttons: hold A + HOME
- LED: Yellow
- Uses Keylink mapping app (QR code in packaging/manual)
Warnings:
- Some games prohibit button mapping; mapping apps may violate game rules.

### Nintendo Switch
- Mode: Switch Controller Mode
- Buttons: hold Y + HOME
- LED: flashing Green
Steps:
1) Turn on Switch
2) Go to Controllers → Change Grip/Order
3) Pair controller (LED flashes Green)

### Windows PC (Windows 10/11)
- Connect via USB-C cable for easiest setup
- Controller should be detected automatically
- Bluetooth pairing can also work via Apple or Android mode depending on PC support

## Features support (if caller asks)
### Turbo
- Enable: Press TURBO, then press the desired button
- Disable: Press TURBO + the same button
- Turbo speed: TURBO + D-Pad Up (faster) / TURBO + D-Pad Down (slower)
Note: Turbo resets when power is turned off.

### RGB lighting
- Press RGB to cycle lighting modes
- RGB + D-Pad Up/Down adjusts brightness
- Hold RGB ~3 seconds to turn lighting off

### Macros (M1/M2)
- Press MACRO + M1 or M2 to start
- Enter button sequence
- Press M1 or M2 again to save
- Clear macro: hold MACRO ~5 seconds

### Vibration strength
- MACRO + D-Pad Up increases vibration
- MACRO + D-Pad Down decreases vibration

## Charging / battery
- Low battery: LED flashes
- Charge via USB-C
- Battery: 3.7V 800mAh
- Play time: ~4–6 hours
- Charge time: ~2.5–3 hours
